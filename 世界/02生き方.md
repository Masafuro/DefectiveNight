# 概要
スキルセットを伴う職業ではなく生き方の系統

# 冒険者
冒険者は、略奪、成長、開拓
成長を止めた時、冒険者は盗賊か虐殺者になるか、静かに隠居するかとなりがち。

# 転生者
異世界等や未来、過去、及び生まれ変わりなどによって転生した者。
属性は、破壊、乾き、未練。
世界へ投げ込まれた楔であり、場合によっては世界そのものを破壊する性質を持つ。
前世の未練と現世で何をしても満たされることがない。

# モグリの魔法士
魔導クランに所属がない魔法士。魔導クランが表に出せない厄介事を処理する暗部と個別に活動する魔法士に分かれる。独自に活動する魔法士をオオカミと呼ぶこともある。

#　落ちこぼれの魔法士
魔法大学から落ちこぼれた者。
しかし、むしろ低位の高速魔法をうまく使うことで冒険者としての適性が高い場合も。
落ちこぼれの魔法剣士

# 七人の魔女
一定水準を超えた魔道士の女の敬称。世界での七人の大魔法使いが七人存在する。
『国が怒らせてはいけない女達』
魔法事故で身体に欠損があることが多い。欠損を隠すため、眼帯やローブなどを重用する。

# 最強
最強の要件は、単に各種のスキルや能力的に強いだけでは十分ではない。何らかの目的に対する狂信的な信念が必要であり、またそのためには手段を選ばない非道さ、容赦のなさを必要とする。現実には様々な誘惑があり、娯楽、男女関係、権力並びに戦闘中やそれ以外における気の配り方において、一貫した集中力を発揮しなければならず、そのためにはそうではない一般人に対して必然的に孤独とならざるを得ない。更には運を必要とし、いくつかの決定的に困難な壁を運よく切り抜けていく中で最強としての自覚と自らの進むべき運命を悟る。
