# 炎の男
あるところに冒険者の男がいた。主に火属性の魔法を得意とし、剣も暑かった。男は冒険者として初めての狩りを終え、なんとかこの土地で冒険者としてやっていけるかもしれないと思っていた。ある時、田舎から幼馴染の女がやってきて冒険者の仲間に加えろといった。男は危ないのでそれはできないと言ったが、女は聞かず、居座ることになった。しばらくすると男も諦めて女を冒険者の仲間に加えることにした。女は水属性の魔法が得意であった。何度かクエストをこなすと、お互いの役割分担も見えてきて、よく魔物を狩れるようになった。男は前に出て剣と炎を放ち、女は水と氷の魔法で男を助けた。男と女は気持ちよくクエストをこなせるようになっていた。
ある時、女が一人で買い物に出かけていたところ、無法者の集団が彼女を攫った。男は宿屋でのんびりしていたが、女が変えるのが遅いので、女を迎えにいくことにした。路地裏に差し掛かったところ、男は見てしまった。女が無法者に■■■■■■■■■■■されているところを。男はたまらず炎の魔法を無法者の集団に向けて放つと、一足に飛び込んだ。手前で男の放った炎にあたった者は焼けて倒れており、それを見た■■■■■■■■■■■していた男と後ろで見ていた男たちは一目散に逃げ出した。男は、■■■■■■■■■■■■■■■■にまみれた女に布切れをかぶせてやると、そっと抱きかかえるようにして宿にもどった。それから男と女の間には、たどたどしい、よそよそしい風が通るようになった。ある時、強大な猪の魔物に直面した。男は一心に魔物に突撃すると、牙で腹を抉られるのもモノともせず、魔物に剣戟と炎を浴びせた。その後、女の氷魔法が大猪を攻撃するはずであった。が、一向に氷の結晶は現れない。男は最後の命を振り絞ると大猪を炎で包み、焼き尽くした。黒く焦げた大きな炭が二人の眼前に横たわった。男は女の方を振り向いた。女はただ立ち尽くしていた。そして、その場に泣き崩れた。
ある時、男が酒場へ入ると、女を■■■■■■■■■■■■■した集団がいた。男は間髪入れず炎を表すと、憎き、殺すべき、その男たちへ向けて極大の炎を放った。無法者の男たちは炭になった。酒場には火の手があがり、客は大声を出して逃げ出し始めた。しばらくすると、王国の騎士達がやってきた。男には、戦う気力はなかったが、全方位を騎士に囲まれ、槍を向けられるとそうも言ってはいられなかった。男が再び、炎を起こそうとした瞬間、騎士達の槍が一斉に男の身体を貫いた。そして自ら生み出した炎がその身を包んだ。男は絶命した。

# 勇者キイチロー
その男の名はキイチロー・ヤマダと言った。男はここではないどこか別の世界からこちらの世界へと召喚されたらしい。当時の国王であったベルエルトⅢ世によって、世界を救う勇者として期待された彼は、しかし、彼が元々持っていた物静かで争いを好まない性格によって、訓練も思うように捗らず、ついには独房に入れられることになった。キイチローが独房で一人静かに佇んでいると、ある者が現れた。その者は、キイチローが以前城下町に出た時に出会った行商人で、怪しげな雑貨を扱っていたと記憶している。その者は言った。「お前さんが牢屋に入れられたと聞き、あまりに可哀想でな。ちょいと助けにきたんだ。」キイチローは驚いた。まさか、街の行商にそのような優しい心が宿っていたとはついぞ思わなかったのである。「ささ。行きますぜ。」キイチローと行商はそそくさと独房を後にした。
キイチローと行商は隣の国までゆくことになった。道中、魔物に襲われたりもしたが、王城で鍛えられた技が役にたった。「ひゅー。さすがは勇者と言われることはありやすね。」行商は茶化した。
隣国につくと宿屋で一泊することにした。キイチローに持ち合わせがないと聞くと行商は気前よく宿代を払ってくれた。キイチローは食堂で夕食をガツガツとかきこんだ。そして、部屋まで案内されるとキイチローは緊張の糸が切れたようにばったりと眠りについた。
なにか、うるさい物音がして目が覚めるとキイチローは驚いた。また、牢屋にいるではないか。昨日までのことは王国の牢屋の中で見た夢だったのだろうか。そうキイチローが思っていると、異変に気がついた。手錠がされていて身動きが取れない。王国ではこうではなかったはずだった。
「ではお集まりの皆様、今日のメインイベント・・・・」
遠くで盛り上がる声が聴こえる。キイチローがいるのは牢というより猛獣用の檻だ。檻ごと、動き出した。そしてステージのようなところに移動させられた。その後のことはよく覚えていない。後で聞いたところによると、キイチローは奴隷として売られたらしかった。しかし、買われた先のディストリア家では、奴隷ではあるものの執事として訓練され、またファミリーを守るための戦闘訓練が施された。またディストリア家はその血も有望視していたらしく、キイチローに何人かの侍女を孕ませた。また執事として仕えており、キイチローを大層気に入っていたディストリアの三女と晴れて結ばれることとなった。後に名家となるヤムド家の誕生であった。

# 聖典天意第一区画
遠い昔、この世界が産まれた始まりの時、世界には何もなかった。そこで神が念じられると二つの星が産まれた。アルト、ベスである。一日目の出来事である。そして神はこの地であるベスを光のある昼と闇のある夜に分けられた。昼には動物が栄えるように、夜には魔物が栄えるようにされた。二日目の出来事である。しばらくすると魔物が闇の力を得て、動物を食い荒らすようになったので、神は人をお造りになられた。そして人にはあらゆる動物と植物、魔物以外の生き物を守るように言い聞かせられた。三日目の出来事である。次第に人が魔物と戦うようになると、昼の世界と夜の世界はお互いに食い合いながらも一つの世界として日々を進むようになった。神はその様子に納得されると、更に遠くの天へと旅立たれた。天上の星が一つまた一つと増え、天上には数多の星が産まれた。四日目の出来事である。そして神がこの世界に息を吹きかけられると、天上のすべての星が少しずつ回転を始め、秩序だった運行をするようになった。五日目の出来事である。神はその様子を見て、納得されると静かに眠りにつかれた。

# 勇者タナカ
世を救った英雄、ないし勇者にも様々あるが、勇者タナカ・タクマほど庶民から好かれている勇者はいないだろう。勇者タナカは王国によって異世界より転生した勇者の一人であるが、転生した時の彼の振る舞いは、後に語られる武勇伝とは程遠く、穏やかで優しいものであったという。

タナカは転生した後、王国の戦士教練へと駆り出されていた。彼は、転生した後、全く何も分からずただ闇雲に、あるいは言われるがままに、ぼんやりと日々の厳しい教練を行っていたという。王国の次女であるアルドネアは度々その様子を見に来ると、教練に気合の入っていないタナカに悪態をついて帰った。
ある時、魔物の進行が明らかになり、戦士団の出立が必要になると、タナカも徴兵されたが、タナカはこれを拒否して逃げ出してしまった。戦士団はそれどころではないため、これを捨て置いたがアルドネアはそうはしなかった。

第二陣の士気は低かった。なにしろ王国の主戦力である戦士団が全滅している。アルドネアは王女として戦闘にたち、戦士たちを鼓舞した。しかし、ここそこで、勇気のない勇者としてタナカはこそこそと噂され、白い目で見られるのだった。

# 花屋アリスタットの娘
とある王国の花屋の娘。その花屋は王室の御用達で、一般的な花だけではなく特殊な花も取り扱っている。例えば、熱暑にした保てない花や寒冷地にしか栄えない花である。こういった遠地の花は、一輪にあってこことは異なる世界を感じさせ、貴族の者たちを中心に非常に好まれていた。このサービスを実現する花屋アリスタットは、創業六十余年となる老舗であり、その花を美しく保つ技術も独自に家系において相伝され今日までその地位を保っていた。扱う魔法は、温湿度のコントロールを主体とするものの、花弁を全く本来と異なる環境で保ち続けるためには、十分な強度と持続時間、既日リズムのコントロールなど極めて繊細な魔法の行使が必要であった。

# 魔王の産まれ
その昔、王国が異世界からの召喚技術を確立した時、ある一人の勇者が召喚された。
勇者の名をアレンという。アレンは当時十数歳であった。
勇者として期待され、魔物の討伐を期待された。
ある時、とある村娘エムリシアと恋仲となり、駆け落ちしてしまった。
王国は勇者を騎士団動員して探し、ついに森の奥地にて二人が生活しているところを発見する。
この時、男の子が一人おり、王国騎士団はこれを取り上げた。
勇者は、人質にされた嫁子のため、手を出すことができなかった。
王国はそのまま勇者の子を王国へと連れて帰った。
勇者と嫁は呆然として立ち尽くしたという。
その後、勇者は王国への根強い恨みから、魔族との交流を持ち始め、次第に頭角を表すと初代魔王となったという。

# 冒険者コディ
コディ・ファルネキは、古くに出版されたものの、今でも最も信頼されているファルネキ魔物辞典の執筆者である。幼少より魔物への興味と観察に才を発揮したコディは、成人の後、冒険者として様々な土地を周り、魔物を観察し記録していた。
コディは魔物辞典の第一巻を出版すると瞬く間に有名になり、恋に政治に大忙しとなった。
しかし、魔物辞典第一巻は、初心者向けの魔物が多く、あくまでビギナーズのための本であり、上級者からは馬鹿にされていた。コディはそのことにも苛立ちを募らせていた。
しかし、元々の趣味である魔物観察は続けていた。ある時、協力な魔物ミノタウロスに遭遇すると、パーティーは自分を除いて全滅してしまった。
ギルドに帰ると、コディはパーティーの仲間を見捨てた生き残りとして悪評が出回り、またコディ自身もそのことを責め、田舎の村に引きこもってしまった。
とある若いパーティーが村を訪れずと、彼に声をかけた。
「うちのパーティーに入ってみないか。」
コディは断った。しかし、何度も訪れる彼の熱意に負けて、また冒険者を始めることになった。
